## :sparkles: Architecture ##

### 详细说明

- **README.md**: 项目说明文档。
- **Framework/**: 源代码目录
  - **main.py**: 主程序文件。
  - **utils/**: 工具模块目录。
    - **helper.py**: 辅助工具模块。
    - **config.py**: 配置文件模块。
  - **models/**: 模型代码目录。
    - **model1.py**: 模型1代码。
    - **model2.py**: 模型2代码。
- **Dataset/**: 测试代码目录。
  - **dataset_for_root_cause.py**: 主程序测试文件。
  - **datset_for_types.py**: 模型测试文件。
- **docs/**: 文档目录。
  - **documentation.md**: 详细文档。
- **requirements.txt**: 项目依赖列表。
- **LICENSE**: 项目许可证。
